### Prerequisites

![Node.js](https://img.shields.io/badge/Node.js-18+-339933?style=flat-square&logo=nodedotjs&logoColor=white)

You will also need:
- NVIDIA API Key (for DeepSeek Chat)
- OpenAI API Key (for optimization suggestions)
- Emergent LLM Key (for unified AI access)

### Installation

#### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/siterank-ai.git
cd siterank-ai
```

#### 2. Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate       # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

#### 3. Configure Backend Environment
Create a `.env` file in `/backend`:

```env
MONGO_URL="mongodb://localhost:27017"
DB_NAME="siterank_ai"
JWT_SECRET="your-secret-key"
NVIDIA_API_KEY="your-nvidia-key"
NVIDIA_BASE_URL="https://integrate.api.nvidia.com/v1"
EMERGENT_LLM_KEY="your-emergent-key"
```

#### 4. Frontend Setup
```bash
cd frontend
yarn install
```

#### 5. Configure Frontend Environment
Create a `.env` file in `/frontend`:

```env
REACT_APP_BACKEND_URL=http://localhost:8001
```

#### 6. Start All Services

```bash
# Terminal 1 â€” MongoDB
mongod

# Terminal 2 â€” Backend
cd backend
uvicorn server:app --host 0.0.0.0 --port 8001 --reload

# Terminal 3 â€” Frontend
cd frontend
yarn start
```

#### 7. Open the App
Visit **http://localhost:3000** in your browser.
